# Portfolio · Leandro Contrera

Sitio web portfolio personal, bilingüe (ES/EN), con dark mode y diseño responsive orientado a desarrolladores.

**Live:** [portfolio desplegado en Cloudflare Pages]

---

## Español

### Tecnologías

- **React 18** + **TypeScript**
- **Vite 6** — bundler y dev server
- **Tailwind CSS 4** — estilos
- **Radix UI** + **shadcn/ui** — componentes accesibles
- **Motion** — animaciones
- **Embla Carousel** — carruseles
- **Lucide React** — iconos

### Secciones

- **Hero** — presentación con redes sociales
- **Sobre mí** — perfil y enfoque profesional
- **Proyectos** — tarjetas con carrusel y modal de detalle
- **Experiencia** — historial laboral
- **Stack tecnológico** — lenguajes, frameworks y herramientas
- **Formación académica** — estudios universitarios
- **Certificados** — certificaciones obtenidas
- **Contacto** — formulario y redes

### Características

- Bilingüe ES/EN con selector en la navbar
- Dark mode por defecto, togglable a light mode
- Diseño mobile-first y totalmente responsive
- Animaciones suaves disparadas por scroll
- Desplegado en Cloudflare Workers con assets estáticos

### Desarrollo local

```bash
pnpm install
pnpm run dev
```

### Build

```bash
pnpm run build
```

La salida se genera en `dist/`.
